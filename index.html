목표를 향해 나아가는 의지와 책임감이 강한 저는 코딩을 할 때 저의 진짜 모습을 만납니다. 특히 생각한 것을 코딩으로 구현해낼 때, 가장 확실한 행복을 느끼고 열정적인 사람이 됩니다. 좋아하기에 더욱 잘하고싶고, 욕심을 내게 됩니다.

좋아하는 일을 잘하는, 그런 개발자가 되고싶은 이유정입니다.

24시간 의료 언택트
서비스 메디플러스

한국 커뮤니케이션 디자인협회 공모전 출품작으로 24시간 의료 언택트 서비스 ‘메디플러스’ 팀 프로젝트입니다. 페이지 단위로 움직이도록 fullPage.js를 사용했으며, 페이지간의 전환과정에서 JavaScript를 사용해 인터랙티브한 요소들을 제어하였습니다. 자연스러운 애니메이션을 구현하는 것을 목표로 제작되었습니다.

  • HTML
  • CSS
  • JavaScript

나이키 브랜드 소개 웹페이지

스포츠 브랜드 나이키를 소개하는 웹 페이지로 나이키의 현장감과 생동감을 표현하기위해 영상 사용하여 풀스크린 형태로 제작하였습니다. 캠페인 영상을 페이지 이동없이 편리하게 볼 수 있도록 JS로 팝업창(모달)을 구현했습니다. 나이키의 심볼과 특징을 살려 css 애니메이션을 구현하였습니다.
